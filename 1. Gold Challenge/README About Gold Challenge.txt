SKILL METRIC :
- Menyusun dokumen test scenario dan test case untuk testing pada website dan aplikasi android.
- Melakukan tipe-tipe testing pada website dan aplikasi android untuk menemukan bug.
- Menyusun bug report untuk setiap bug yang ditemukan pada website dan aplikasi android.
- Melakukan API testing.
- Melakukan Performance testing.

DELIVERY :
- Mempelajari flow website dan aplikasi android.
- Menyusun bug report sesuai bug/defect yang ditemukan dengan menggunakan severity Critical-Major-Medium-Minor dan priority High-Middle-Low.
- Mengkonfigurasi Postman untuk melakukan API testing berdasarkan dokumentasi API dari BE.
- Mengkonfigurasi JMeter untuk melakukan Performance testing berdasarkan dokumentasi API dari BE.

CRITERIA :
- Test scenario dan test case terdokumentasi dalam Microsoft Excel sesuai dengan format (25%).
- Bug report terdokumentasi dalam Microsoft Word sesuai dengan format (25%).
- Collection, Request, konfigurasi Headers, konfigurasi Body, konfigurasi Test di Postman dibuat berdasarkan dokumentasi API dan di-export ke dalam format JSON (15%)
- API testing report dibuat menggunakan Newman dan disimpan dalam bentuk .html (15%)
- JMeter dikonfigurasi dan performance test dilakukan dan disimpan dalam bentuk .jmx (10%)
- Performance testing report dibuat menggunakan JMeter dan disimpan dalam bentuk .html (10%).
